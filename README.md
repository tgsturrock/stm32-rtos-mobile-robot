# STM32 RTOS Mobile Robot

A mobile robot developed on an STM32F051R8 microcontroller using FreeRTOS. The project demonstrates real-time task scheduling, peripheral driver development, and closed-loop motor control for an autonomous embedded system.

## Features

- FreeRTOS task scheduling
- PWM motor control
- ADC sensor acquisition
- UART communication
- I2C peripheral communication
- Timer and interrupt handling
- State-machine based robot control
- Real-time control loop

## Hardware

- STM32F051R8
- DC motors
- Infrared distance sensors
- Wheel encoders
- OLED display
- User buttons and LEDs

## Software

- C
- FreeRTOS
- STM32CubeIDE
- STM32 HAL

## Concepts Demonstrated

- Real-time embedded systems
- RTOS task scheduling
- Peripheral driver development
- Hardware abstraction
- Interrupt-driven programming
- Embedded software architecture
